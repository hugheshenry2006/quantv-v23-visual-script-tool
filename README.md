## What QuantV Does

QuantV is a visual preset made for GTA V users who want a more controlled and polished image output. Its focus is on color shaping, lighting refinement, sharper detail, and adjustments to shadows, contrast, and scene tone.

The preset is meant for both single-player and multiplayer client environments, including FiveM, RageMP, and alt:V. It also provides ReShade preset support together with timecycle control, so the game can be tuned visually without altering gameplay itself.

## Features

- Broad visual tuning through color and lighting enhancement
- Timecycle control for changing atmosphere and environment feel
- Sharpness adjustments for a clearer image
- Shadow and contrast controls for balancing scene depth
- ReShade preset support for compatible configurations
- Workflow oriented toward GTA V visual mod setups
- Suitable for both single-player and multiplayer clients
- Intended for FiveM, RageMP, and alt:V environments

## Compatibility Notes

QuantV is designed for GTA V and client-based multiplayer setups like FiveM, RageMP, and alt:V. It can also be used in single-player environments when the preset format is supported.

Actual results can differ based on the game build, client restrictions, installed shader stack, and any other visual mods already present. When more than one preset or shader package is active, some settings may overlap or need manual correction.
